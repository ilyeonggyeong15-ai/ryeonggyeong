## 메인화면
- 검색창 : 카테고리로 검색할 수도 있고, 원하는 음식점을 검색할 수도 있다.
- 카테고리 : 한식, 일식, 중식, 분식 등
- 필터 : 거리순, 리뷰순, 가격순
- 밥집 리스트 : 오른쪽 스크롤을 통해 위아래로 넘기며 원하는 음식점 찾기 

## 상세화면
- 가게 사진 : 좌우 슬라이드 형식으로 넘기며 최대 3개의 사진을 볼 수 있다.
- 위치 : 네이버 지도 API 사용
- 영업시간 : 가게 별 영업시간을 표시
- 리뷰목록 : 인기순, 최신순으로 정렬하고, 리뷰 사진만 몰아볼 수도 있다.

## 리뷰 작성화면
- 이전 화살표 : 리뷰 작성하는 화면으로 들어왔을 때 왼쪽 상단 화살표를 통해 상세화면으로 돌아가기
- 사진 업로드 : 업로드 할 수 있는 사진은 최대 3장
- 리뷰작성칸 : 띄어쓰기를 포함하여 최대 20자 이상 작성하기
- 재방문 의사 확인 체크 : 재방문할 의사가 있는지 없는지 판단하는 체크 표시 버튼
- 예시 리뷰 : "혼밥하기 좋아요.", "양이 많아요." 등
- 등록하기 : 등록버튼을 누르면 욕설이 있는지 없는지 판단한 뒤 등록해줘
